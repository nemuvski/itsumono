**isString**(`value`): value is string

引数valueがstringの場合はTrueを返却

**`Example`**

```ts
let value = getValueFn()
if (isString(value)) {
  // この時、valueはstring値と推論できる
}
```

## Parameters

| Name | Type |
| :------ | :------ |
| `value` | `unknown` |

## Returns

value is string
