**isNumber**(`value`): value is number

引数valueがnumberの場合はTrueを返却

**`Example`**

```ts
let value = getValueFn()
if (isNumber(value)) {
  // この時、valueはnumber値と推論できる
}
```

## Parameters

| Name | Type |
| :------ | :------ |
| `value` | `unknown` |

## Returns

value is number
